# Hobby Hub

**Hobby Hub** é um aplicativo Android desenvolvido em **Kotlin**, criado como projeto da disciplina **Programação para Dispositivos Móveis** do **Instituto Federal de Alagoas (IFAL) - Campus Arapiraca**, ministrada pelo professor **Tarsis Marinho** durante o período acadêmico de 2025.2.

O app reúne diferentes **hobbies**, sendo cada um deles desenvolvido por um dos integrantes do grupo, resultando em uma coleção diversificada de funcionalidades e experiências dentro de um único aplicativo.

## Sobre o Projeto

O **Hobby Hub** é uma aplicação modular onde cada integrante ficou responsável por implementar um hobby diferente, explorando conceitos de integração com APIs, armazenamento local e boas práticas de arquitetura Android (ou pelo menos tentamos).

O projeto segue a estrutura padrão do Android Studio e utiliza o **Gradle** para gerenciamento de dependências e automação de build.

## Tecnologias e Bibliotecas Utilizadas

- **Kotlin** - Linguagem principal do projeto
- **Jetpack Compose** - Framework declarativo de UI utilizado em toda a interface
- **Android Studio** - IDE de desenvolvimento
- **Gradle** - Sistema de build e gerenciamento de dependências

### Bibliotecas Principais

- **Coil** - Carregamento e exibição eficiente de imagens
- **Retrofit** - Comunicação com APIs REST
- **Room** - Banco de dados local com ORM integrado ao Jetpack

## Como Executar o Projeto

1. **Clone este repositório:**
   ```sh
   git clone https://github.com/Maheshivara/HobbyHub.git
   ```
2. Abra o projeto no Android Studio:
    1. Vá em File -> Open

    2. Selecione a pasta HobbyHub

3. Sincronize o Gradle:

    - O Android Studio faz isso automaticamente ao abrir o projeto

> [!WARNING]
> Caso não, vá em File -> Sync Project with Gradle Files

4. Execute o aplicativo:
    1. Escolha um emulador Android ou dispositivo físico
    2. Clique em Run
